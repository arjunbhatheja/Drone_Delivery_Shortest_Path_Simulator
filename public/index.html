Drone Movement Simulator

Assume that you only have 1 remote to control all the drones, my project calculates the minimum moves that would take to deliver drones to their respective locations given there are also no fly zones in the middle. You can set up the board for which you want to find the minimum moves. MAX Number of Drones could be only 4. Some Requirements -
  • Drones can't fly over the destination of another drones
  • Drones can't fly over the Fly Zone
  • You can use arrows to try to get the drones to their respective locations in minimum moves
  • Click on Calculate moves and wait for 3 seconds to start the shortest path simulation
